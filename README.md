* Limitações das ferramentas SAST (apenas detectam).
* Proposta: Usar ML para sugerir/realizar correções automatizadas.

![Ícone de Machine Learning para segurança de código](icon_ml_code_security.png)

### Slide 5.2: Metodologia e Avaliações - Construindo e Validando a Solução de ML
